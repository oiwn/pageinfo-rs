## Working on this project

- Run `cargo fmt` after finishing any task.
- `cargo clippy --all-targets -- -D warnings` must pass — fix before stopping.
- `cargo test` after every change.
- Keep responses short — fit a single screen. No walls of text.
- View file before editing. Match whitespace exactly.
- No comments in code unless asked.
